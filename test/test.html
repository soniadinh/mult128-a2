Bootstrap Class Reference

Layout

  • container: Centers content and sets max-width based on screen size
  • row: Creates a horizontal flex container for columns
  • col-md-4: 3 equal columns on medium screens and up
  • g-3: Adds spacing (gutters) between columns

Spacing

  • py-5: Padding top and bottom
  • mb-3, mb-4: Margin bottom
  • mt-4: Margin top
  • p-3, p-2: Padding on all sides

Colors

  • bg-light: Light background
  • bg-dark: Dark background
  • bg-primary, bg-secondary: Theme colors
  • text-white: White text

Typography

  • display-1, display-4: Large headline styles
  • lead: Emphasized paragraph
  • fw-bold: Bold font weight
  • fst-italic: Italic text
  • lh-lg: Larger line height
  • text-center: Center text
  • text-decoration-underline: Underlined text

Flexbox

  • d-flex: Enables flexbox
  • justify-content-between: Spaces items apart

Images and Utilities

  • img-fluid: Responsive image
  • rounded: Rounded corners
  • shadow, shadow-sm: Adds shadow
  • border: Adds border

Components

  • btn: Button base
  • btn-primary, btn-secondary: Button styles
  • card, card-body: Card layout
  • nav, nav-tabs: Navigation styles
  • pagination: Pagination component
  • progress, progress-bar: Progress bars
  • spinner-border: Loading spinner
  • badge: Small label indicator
  • breadcrumb: Breadcrumb navigation

Interactive (JS-based)

  • data-bs-toggle="collapse": Toggles hidden content
  • data-bs-toggle="modal": Opens modal
  • data-bs-toggle="dropdown": Dropdown menu
  • accordion: Expandable sections
  • alert: Notification box

Body Attributes

  • data-bs-spy="scroll": Enables scrollspy
  • data-bs-target="#navbar": Links scrollspy to navbar
  • data-bs-offset="50": Offset for scroll tracking
